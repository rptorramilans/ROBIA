# ROBIA
The spatial distribution of a dye tracer is evaluated from UAV-based RGB imagery, processed through a robust and systematic approach developed to detect rhodamine dye presence and estimate its concentration at the water surface.
Code developer: Raquel Peñas Torramilans from Laboratori d'Enginyeria Marítima (LIM-UPC), Universitat Politècnica de Catalunya (UPC-BarcelonaTech), raquel.penas@upc.edu
